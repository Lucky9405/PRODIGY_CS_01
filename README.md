# Caesar Cipher Encryption & Decryption

## Task 01 - Cyber Security Internship @ Prodigy InfoTech

### Overview

This project implements the **Caesar Cipher Algorithm** in Python. The program allows users to encrypt and decrypt text messages by specifying a shift value.

### What is Caesar Cipher?

The Caesar Cipher is one of the oldest and simplest encryption techniques. It works by shifting each letter in a message by a fixed number of positions in the alphabet.

**Example (Shift = 3):**

* A → D
* B → E
* C → F

Message:
HELLO

Encrypted:
KHOOR

To decrypt the message, the letters are shifted back by the same value.

### Features

* Encrypt text using Caesar Cipher
* Decrypt encrypted text
* User-defined shift value
* Supports uppercase and lowercase letters
* Preserves spaces and special characters
